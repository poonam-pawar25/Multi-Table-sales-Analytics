# Multi-Table-sales-Analytics
Developed an interactive Sales Performance Dashboard using Power BI by integrating multiple datasets

#Project Overview

This project focuses on building an interactive Sales Performance Dashboard using Power BI. The dashboard integrates multiple datasets to analyze sales performance and generate meaningful business insights for better decision-making.

#Objectives

-Analyze overall sales performance
-Track customer and product trends
-Monitor employee-wise sales contribution
-Provide interactive reporting for business insights

#Dataset Details

The dashboard is built using multiple tables:
Customers – Customer details and segmentation
Products – Product information and categories
Employees – Sales representatives data
Sales – Transactional sales data
Date – Time-based analysis

#Tools & Technologies Used

Power BI
DAX (Data Analysis Expressions)
Data Modeling (Relationships )

#Key Features

-Interactive dashboard with filters and slicers
-Drill-down and drill-through capabilities
-Data model with relationships across multiple tables
-KPI tracking (Total Sales, Revenue, etc.)
-User-friendly and dynamic visualizations

#Key Insights

Identified top-performing products contributing maximum revenue
Analyzed customer purchasing patterns and trends
Evaluated employee-wise sales performance
Observed monthly and yearly sales growth trend
